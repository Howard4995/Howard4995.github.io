關於 Python 線上執行器

這個工具使用 Pyodide 在您的瀏覽器中執行 Python 程式碼，無需後端伺服器。

支援功能：

  • Python 3 標準語法
  • 大部分標準庫
  • 基本的 NumPy、Pandas 等套件
  • Matplotlib 圖表 (需要額外啟用)

限制：

  • 無法訪問本地檔案系統
  • 執行速度可能比本機安裝的 Python 慢
  • 可能無法使用一些依賴特定平台的套件
提示： 您可以使用這個工具來學習和測試 Python 程式碼。如果您需要更多功能，請考慮在您的電腦上安裝完整的 Python 環境。
